# nihonGO
### Purpose
Personal project for learning go and studying nihongo. 

For now I'm focused on the flashcards modes but I plan to work on the advanced modes soon.

### Planned features
- kana flashcard
    - hiragana mode
    - katakana mode
    - inverted mode(show romaji and asks for kana)
- word flashcard
    - kana mode
    - kanji mode
- word type
    - kana mode
